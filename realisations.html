Irigo Mobile

Irigo est le transporteur urbain dans la région d'Angers, j'ai réalisé l'application sur iOS.

Cette application permet aux utilisateurs de pouvoir consulter :

  • Les horaires des différents bus.
  • Le prochain passage d'un bus sur un arrêt particulier
  • Effectue un itinéraire entre deux arrêts
  • Affichage des différents plan des bus (Journée, nuit, vacances, ...)
  • Affichage des informations importantes (Travaux, nouveautés, ...)
  • Client: Irigo
  • Date: Décembre 2012
  • Service: Dévelopment iOS

WatchServ

WatchServ est une application sur Android, elle permet la supervision de différents serveurs.

Elle envoie une notification sur le mobile lorsqu'un des serveurs à un indicateur au dessus de certains seuils.

  • Client: Altaïr Multimédia
  • Date: Février 2013
  • Service: Développement Android

Promocadeaux

Suza

Suza est un site spécialisé dans la vente de matériels High Tech présent dans la région d'Angers.

La cible d'utilisateurs visée de ce site sont des revendeurs ou futur revendeur de leurs produits.

Le site permet aux revendeurs de voir les différents produits de la société Suza, et de voir les prix qui lui sont appliqués.

  • Client: Suza
  • Date: Juin 2014
  • Service: Développement Web / Framework Zend
